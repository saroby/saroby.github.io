정렬

삽입 정렬

자료 배열의 모든 요소를 앞에서부터 차례대로 이미 정렬된 배열 부분과 비교하여, 자신의 이치를 찾아 삽입함으로써 정렬.

선택 정렬

자료 배열의 모든 요소를 차례대로, 주어진 리스트 중에 최소값을 찾은 후 리스트의 맨 앞의 값과 교체함으로써 정렬.

버블 정렬

자료 배열의 사이즈만큼 순환하되, 각 순환에서 인접한 수들을 비교하여 교체하는 방식으로 정렬. 순환 할때마다 최소 혹은 최대값이 하나씩 정해지므로 내부 비교 수는 점점 줄어든다.

합병 정렬

자료 배열을 1/2로 반복하여 쪼개어 각 1개의 수를 가지도록 한 후, 쪼갠 수들을 합병하면서 정렬을 한다. 합병에 사용되는 2개의 리스트들의 맨앞에 위치한 값들 중 작은 값을 하나씩 사용함으로써 하나의 리스트를 만들어간다. 마지막에 하나의 리스트가 만들어지면서 정렬이 완료.

퀵 정렬

자료 배열에서 하나의 원소를 고르고 이를 피벗으로 부른다. (피벗을 제외한 리스트에서) 왼쪽에서는 i위치값이 증가하면서 피벗보다 큰 수를 찾고, 오른쪽에서는 j위치값이 감소하면서 피벗보다 작은 수를 찾는다. i와 j 모두가 수를 찾으면 값을 교환하고, 값을 찾는 과정 반복한다. i의 위치가 j의 위치보다 커지면 피벗값과 i의 값을 교환한다. 피벗은 정렬이 완성됐으므로 피벗의 왼쪽과 오른쪽을 자료 배열로 하여 재귀적으로 같은 방법을 수행함으로써 정렬.

기수 정렬

가장 낮은 자리 수에서 정렬을 하고 자리 수를 높은쪽으로 옮겨 가면서 정렬을 하면 정렬이 완료.
